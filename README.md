# 🤖 Chatbot Project

Live Demo: https://selty-tech.github.io/chatbot-project/

---

## 📌 About This Project

This chatbot application was built as part of my learning journey in frontend development.

The base project was created by following a React tutorial (SuperSimpleDev) to understand core concepts such as components, state management, and user interaction.

After completing the tutorial, I focused on modifying and improving the project to better understand how the code works and how features can be extended.

---

## 🚀 Features

### Core functionality (from tutorial)
- Send and receive chatbot messages
- Basic chatbot responses
- Persistent chat using localStorage

### Improvements & adjustments
- Dynamic date response (real-time instead of static value)
- Sending messages using the Enter key
- Auto-scroll to the latest message
- Typing/loading indicator for better user experience
- Clear chat functionality with confirmation
- Improved input handling and validation
